Успешная регистрация 1.1.1
Предусловия:
  - Пользователь не зарегистрирован на сайте
  - Выбран интерфейс на русском языке (не требуется при переключении на селекторы)
Шаги:
- Шаг 1, заполнить поле `name["registration-email"]`
- Шаг 2, заполнить поле `name["registration-password1"]` сложным значением, больше 9 символов, использующем разные классы
  символов
  проверка: проверить значение в поле `name["registration-password1"]`
  ожидаемый результат: в поле отображаются символы "*" по количеству символов в пароле
- Шаг 3, заполнить поле `name["registration-password2"]` тем же значением, которое было использовано на шаге 2
  проверка: проверить значение в поле `name["registration-password2"]`
  ожидаемый результат: в поле отображаются символы "*" по количеству символов в пароле
- Шаг 4, нажать на кнопку "Зарегистрироваться"
  проверка: проверка контента страниц
  ожидаемый результат:
    - на странице присутсвует блок ".alert.alert-success" с текстом "спасибо за регистрацию"
    - присутствует элемент `ul.nav.navbar-nav.navbar-right a i.icon-signout`
    - присутствует элемент `ul.nav.navbar-nav.navbar-right a i.icon-user`

Логин зарегистрированного пользователя 1.2.1
Предусловия:
  - Пользователь зарегистрирован на сайте
Шаги:
- Шаг 1, заполнить поле `name["login-username"]` значением email
- Шаг 2, заполнить поле `name["login-password"]` значением пароля
  Проверка: проверить отображающиеся в поле символы
  В поле должны отображаться символы: *
- Шаг 3, нажать на кнопку `name["login_submit"]`
  Проверка: проверить контент страницы
  Ожидаемый результат: 
    - присутсвует блок `.alert.alert-success` с текстом "Рады видеть вас снова"
    - присутствует элемент `ul.nav.navbar-nav.navbar-right a i.icon-signout`
    - присутствует элемент `ul.nav.navbar-nav.navbar-right a i.icon-user`

Увеличение количества товара 2.1.1
Предусловия:
  - в корзине есть заказанный товар
Шаги:
  - Шаг 1, найти элемент `name["form-0-quantity"]`
    Ожидаемый результат элемент существует
  - Шаг 2, установить фокус в элемент `#id_form-0-quantity`
  - Шаг 3, стереть значение и ввести новое (10)
  - Шаг 4, нажать на кнопку "обновить" `#id_form-0-quantity + span > button`
    Проверка: 
      - число в поле `#id_form-0-quantity`
    Ожидаемый результат:
      число равное введённому на шаге 3
    Проверка: значение "всего" в ряду
    Ожидаемый результат:
      значение "всего" в ряде равно quantity*price
